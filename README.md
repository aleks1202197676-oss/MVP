# MVP Hub

## Быстрый старт
```bash
python -m pip install -r requirements.txt
python -m pipelines.run_all
```

После успешного прогона будут обновлены:
- `contracts/meta/COMPILED_MEMORY.md`
- `contracts/meta/project_state.yml`
- `data/ai_contract/latest/latest_bundle.zip`

## Передача проекта в новый чат
Откройте корневой `START_HERE.md`, затем следуйте ссылке на `contracts/meta/START_HERE.md`.

