# Glossary

- **Memory Pack** — набор файлов в `contracts/meta` для передачи контекста проекта.
- **LLM bundle** — zip-архив с данными и контрактами для анализа в LLM.
- **run_manifest.json** — метаданные последнего прогона и список файлов bundle.
- **latest_bundle.zip** — актуальный архив, который прикладывается в новый чат.
- **finance templates** — CSV-шаблоны для ручного ввода и импорта финансовых данных.
