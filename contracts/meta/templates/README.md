# Templates

Папка содержит шаблоны CSV для финансовых входов.

## Как использовать
1. Скопируйте нужный `*.template.csv` в рабочую директорию данных.
2. Переименуйте файл, убрав `.template`.
3. Заполните строки, не меняя заголовки колонок.
4. Используйте UTF-8 и разделитель `,`.

## Доступные шаблоны
- `finance/cards.template.csv`
- `finance/items.template.csv`
- `finance/budget_daily.template.csv`
- `finance/manual_payments.template.csv`
