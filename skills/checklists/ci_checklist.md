# Checklist: CI Quality Gate

- [ ] Проверки детерминированы (без зависимости от текущего времени/locale).
- [ ] Ошибки fail-fast и с actionable diagnostics.
- [ ] Нет лишнего шума в логах.
- [ ] Невалидный кейс стабильно падает.
- [ ] Валидный кейс стабильно проходит.
- [ ] Правила deny/allow документированы рядом с проверкой.
