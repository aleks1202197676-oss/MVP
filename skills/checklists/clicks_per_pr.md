# Checklist: Clicks per PR

Цель: уменьшить ручные действия и повторяемые ошибки в процессе ревью/доставки.

- [ ] Описание PR содержит готовые команды проверки (copy-paste).
- [ ] Есть короткий список changed files по смысловым блокам.
- [ ] CI комментарии и сообщения ошибок не требуют "угадывания" следующего шага.
- [ ] Если процесс повторяется, добавлен/обновлён шаблон в `skills/`.
- [ ] Минимум ручных кликов: одна явная последовательность действий от review до merge.
