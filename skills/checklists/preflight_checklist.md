# Checklist: Preflight

- [ ] Определён точный scope файлов (что можно/нельзя менять).
- [ ] Найдена точка входа и основной кодовый путь.
- [ ] Сформулированы acceptance criteria (измеримые).
- [ ] Подготовлен список команд проверки.
- [ ] Описаны top-5 failure modes + mitigation.
- [ ] После preflight получено явное `ok` на реализацию.
